### Lab41 Loop: Lab2
code ทั้งสองอันนี้จะแสดง alert ออกมาเหมือนกันทั้งหมดหรือไม่  
หริณ มาเบ้า
```Javascript
let i = 0;
while (++i < 5) {
  alert( i);
}
```
++i คือ +1ก่อนประมวลผล
+1 >> คำนวณ >> แสดงผล จะได้ 1 2 3 4 (5 > 5 ได้ false จึงไม่แสดง)

```Javascript
let i = 0;
while (i++ < 5) {
  alert( i);
}
```
i++ คือ ประมวลผลเสร็จค่อย +1
